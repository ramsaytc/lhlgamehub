# LHL Game Hub

A modern, mobile-friendly web app for browsing **Lakeshore Hockey League (LHL)** game scores and standings.

This project combines:
- **Automated scraping** of LHL game schedules, results, and standings
- **Data normalization** into clean CSV and JSON files
- A **Next.js + React** frontend with fast search, autocomplete, dark mode, and responsive design
